Place your console icon images here. Expected filenames (used by the app):

  gamecube.png   - GameCube
  wii.png        - Wii
  ps1.png        - PlayStation 1
  ps2.png        - PlayStation 2

Icons appear to the left of each game name. Use square or near-square images;
they are shown at 32px (28px on mobile). If a file is missing, that slot is left blank.
